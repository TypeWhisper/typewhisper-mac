TypeWhisper 1.2.3 is a stable maintenance release focused on fixing OpenAI GPT-5 prompt processing when the bundled OpenAI / ChatGPT plugin uses API-key authentication.

## Highlights

- **GPT-5 API-key compatibility restored** - the bundled OpenAI plugin now omits `temperature` for GPT-5 chat-completions requests when reasoning effort is enabled, which matches the current OpenAI API requirements.
- **Safer OpenAI chat helper controls** - the plugin SDK helper can now omit `temperature` entirely for providers and models that reject explicit overrides.

## Fixes & Compatibility

- Fixed `Unsupported value: 'temperature'` failures for `gpt-5.4` and related GPT-5 API-key flows in the bundled OpenAI / ChatGPT plugin.
- Added regression coverage for GPT-5 temperature handling in both the SDK helper tests and the host app plugin tests.
- Preserved `PluginOpenAIChatHelper` ABI compatibility so existing external plugin bundles continue to load after the SDK helper extension.
